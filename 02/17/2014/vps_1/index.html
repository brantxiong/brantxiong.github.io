增加非root用户

当以root权限登陆一台全新的VPS之后，第一件事情就是增加拥有sudo权限的非root用户，目的如下:

  • it prevents the user making any system-destroying mistakes
  • it stores all the commands run with sudo to a file where can be reviewed later if needed
