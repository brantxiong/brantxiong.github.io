基本用法

tcpdump 采用命令行方式，需以root权限运行，它的命令格式为：

tcpdump [ -adeflnNOpqStvx ] [ -c 数量 ] [ -F 文件名 ]
        [ -i 网络接口 ] [ -r 文件名] [ -s snaplen ]
        [ -T 类型 ] [ -w 文件名 ] [表达式 ]

简单用法：

tcpdump -i eth1 port 8001
#查看localhost数据包
tcpdump -i lo port 8001

将结果写入文件：

tcpdump -i lo -nn port 8001 -w dump

参数说明：

-d 启动一个守护进程
-p 端口
-m 分配的内存是M
-c 最大运行并发数
